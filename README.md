# CLPsych.org Website

## Updating the Website

To update the website, make changes to the markdown files in `templates`.

The website will build and publish to GitHub Pages when a commit is made.

## Building Locally (optional)

Youc an build the site by running `make`. You may need to first install dependencies with `pip install -r requirements.txt`.
